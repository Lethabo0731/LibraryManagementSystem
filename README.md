# Library Management System

A simple **Library Management System** built in **Java** using **JDBC** and **MySQL**.  
This project allows users to manage books, users, and transactions (borrow/return books) with a database backend.

---

## Features

- Add, view, and manage books  
- Add and manage library users  
- Borrow and return books  
- Track transactions in the database  
- CRUD operations implemented with JDBC  
- Database: MySQL
